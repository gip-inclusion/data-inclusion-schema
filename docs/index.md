# Le commun numérique de l’offre d’insertion

**data·inclusion recense les services d'insertion sociale et professionnelle en France.**

Cette page vous permettra de vous familiariser avec les différents aspects de data⋅inclusion.

## API

L’API permet d’accéder aux contacts associés aux structures et services, à la [déduplication](https://data.inclusion.gouv.fr/outils-et-documentation/blog-datainclusion/), au [score de qualité](https://data.inclusion.gouv.fr/outils-et-documentation/blog-datainclusion/), à des filtres avancés, et à des mise à jour quotidiennes.

C'est le moyen privilégié pour des intégrations avancées du jeu de données data⋅inclusion.

Pour accéder à la documentation de l’API :

- En [environnement de pré-prod](https://api-staging.data.inclusion.beta.gouv.fr/api/docs)
- En [environnement de production](https://api.data.inclusion.beta.gouv.fr/api/docs) - un jeton sera nécessaire
- Le [formulaire de demande de jeton API](https://tally.so/r/mYjJ85)

## Widget
data⋅inclusion propose un [widget](https://data.inclusion.gouv.fr/acc%C3%A9der-aux-donn%C3%A9es/widget/) personnalisable permettant l'exploration du jeu de données sur votre site avec un effort d'intégration minimal; une seule ligne de code suffit.


## Jeux de données
- Un [tableau de bord](https://data.inclusion.gouv.fr/outils-et-documentation/aper%C3%A7u-de-nos-donn%C3%A9es/) est disponible pour avoir un aperçu avec graphiques récapitulatifs des données disponibles
- Un [export de données brutes](https://data.inclusion.gouv.fr/acc%C3%A9der-aux-donn%C3%A9es/telechargement-de-nos-donnees/) permet de télécharger nos données directement
- La base consolidée est aussi publiée, aux formats json et csv, sur [data.gouv.fr](https://www.data.gouv.fr/en/datasets/referentiel-de-loffre-dinsertion-liste-des-structures-et-services-dinsertion/).

## Schéma data⋅inclusion
data⋅inclusion a établi une structure de données précise pour les objets suivants:

- [Structure](structure.md) : une structure proposant des services d'insertion
- [Service](service.md) : un service d'insertion
